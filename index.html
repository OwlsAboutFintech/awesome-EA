我要怎麼用這些 EA?
EA 是交易平台 MetaTrader 的交易程式的稱呼，多數國際外匯與原物料交易券商都有提供下載，帳號辦了就有（也可以開模擬帳號）然後就可以使用本站的所有 EA。

如何在我的 EA 添加市場輪廓?
如果你想在你自己寫的 EA 添加市場輪廓模組，請參考與閱讀 這個 GitHub 專案 並依照步驟加入到你的 EA 程式碼。

怎麼用市場輪廓輔助手動交易?
你可以在 這裡 下載到市場輪廓的技術指標，不一定要將市場輪廓加到 EA 也可以幫助你的交易。

 
快速開始

預先要求：準備好 MetaTrader 4/5 的交易環境
請設定好你的環境，並且依照下列步驟啟動 EA：
  • 將 EA/ 資料夾放到 MQL4/MQL5 > Experts 目錄，打開 EA.mq4/EA.mq5 並按編譯。
  • 在交易視窗中掛上 EA 或在回測面板中選定 EA，並打開參數面板讀取參數檔案 params.set
步驟一：回測與驗證
  • 關於 MetaTrader 回測方面，網路上有很多工具可以協助回測得更加正確與精準，請自行研究。
  • 本站提供的 EA 回測報告僅在基礎的 MetaTrader 回測功能上以 Tick 資料在券商 ICMarket 上進行浮動 Spread 之回測
  • 在同樣回測條件下照理來說不應該差異到非常大，也歡迎回報結果相差甚遠的 EA
  • 總之有問題就是討論就對了，如果你願意一起交流
步驟二：前測與使用
任何的 EA 除了回測，更重要是前測（Forward Testing）推薦你應該要以模擬帳戶與真實帳戶進行各自一週的測試，本站程式與 EA 僅供學習與研究之用途，如用於真實環境獲利盈虧請自行負擔，本站不負責任何風險。
